## Descrição
Projeto backend desenvolvido com o framework [Nest](https://github.com/nestjs/nest).

## Instalação
```bash
# backend
$ cd ./backend && yarn

# frontend
$ cd ./frontend && yarn
```

## Executar

```bash
# banco de dados
$ yarn docker

# backend
$ yarn start:api

# frontend
$ yarn start:front
```

## Testar

```bash
# backend
$ yarn test

# frontend
$ yarn test:front
```
